For datasets that returns json arrays, you can choose the field that will be used as values to autocomplete in the field. You can use just the name of the field or you can write javascript expressions between {} that will compose the values of the json objects to compose the value.
If the dataset returns a simple list of strings, the value of this field is ignored.
Some examples. Let's assume the dataset resulted in the following json:
[
	{"name": "Eddard Stark","id":"estark", "house":"Stark"},
	{"name": "John Snow","id":"jsnow", "house":"Stark"},
	{"name": "Tyrion Lannister","id":"tlannister", "house":"Lannister"},
	...
]
Some valid examples:
  • name
  • house
  • {$name}
  • {$name+" ("+$house+")"}
